
# 数据结构 ：逻辑结构、物理结构
```java
/**
 
         逻辑结构
             集合结构（元素之前没有其他关系）
             线性结构（元素之间一一对应线性关系）
             树形结构（元素之间一对多线性关系）
             图形结构（元素之前多对多关系）
         
         物理结构 ：逻辑结构在计算机中真正的表现方式，也叫存储结构
             顺序存储结构（数据元素存放的地址是连续的）
             链式存储结构（数据元素存放地址可以是不连续的）

* 
**/


```
